简介

dify-spring-boot-starter 是一个 基于 springboot 实现的 Dify 接口调用 框架。

特性

  • 无侵入: 通过 spring-boot-starter的方式，无侵入式的集成调用。
  • 统一规范： 基于Dify 接口 规范。

支持的服务

  • Chat (聊天相关)
  • Workflow (工作流相关)
  • Dataset (知识库)
  • Server (没提供开放接口的功能)
